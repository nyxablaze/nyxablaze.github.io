Setting up Server

STEP 1: ARCH INSTALLATION

Installation Guide

When using PacStrap, include extras:
• git
• networkmanager
• yay
• xorg
• xorg-xinit
• i3-wm
• i3lock
• i3status
• i3blocks
• dmenu
• terminator
• firefox
• sudo
i.e. pacstrap command will look like this:
pacstrap -K /mnt base linux linux-firmware git networkmanager yay xorg xorg-xinit i3-wm i3lock i3status i3blocks dmenu terminator firefox sudo
Ensure you are in the sudoers group, so you can use command sudo successfully.
